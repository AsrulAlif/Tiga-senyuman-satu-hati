# Program_Skripsi

“Tiga Senyuman Satu Hati” adalah sebuah game visual novel yang berlatar di SMA3 Gerberas, sebuah sekolah menengah atas dengan kehidupan sosial yang dinamis. Pemain berperan sebagai siswa pindahan kelas 2 SMA yang ingin memulai hidup baru, mencari teman sejati, dan menemukan cinta pertamanya. Sepanjang cerita, Pemain akan bertemu dengan tiga karakter perempuan yang masing-masing memiliki kepribadian serta latar belakang unik, ditambah satu hidden character dan terdapat satu lagi NPC untuk menambah kan cerita di awal game.

Manual: https://drive.google.com/file/d/17NEu6plBwhgyKbTu5aNixCbyx7mY8OmK/view?usp=sharing

Manual ini mencakup tautan unduhan, cara mengunduh, cara menginstal, cara bermain, dan semua informasi tentang spesifikasi dan permainan.
