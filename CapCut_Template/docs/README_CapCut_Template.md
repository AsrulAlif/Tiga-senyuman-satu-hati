# CapCut Template - Tiga Senyuman Satu Hati

Template ini dibuat untuk trailer visual novel berdurasi 45 detik dengan rasa awal yang manis, lalu berubah jadi misterius/glitch.

## Cara Pakai Di CapCut

1. Buka CapCut dan buat project baru 16:9.
2. Import semua file dari folder `CapCut_Template/media`.
3. Taruh media di timeline sesuai urutan nama file `01` sampai `12`.
4. Atur durasi tiap media mengikuti `timeline_capcut.csv`.
5. Import subtitle `docs/captions_id.srt` lewat menu Captions/Subtitles.
6. Tambahkan musik: mulai ceria pada 00:00, lalu stop/hit pada 00:20.5, kemudian masuk ambience gelap.
7. Pakai efek yang ditulis di kolom `CapCut effect`.

## Style Yang Dicari

- Awal: pink/romance, cute, visual novel, slow zoom.
- Tengah: karakter muncul satu-satu dengan dialog pendek.
- Twist: cut mendadak ke kartu gelap `TAPI...`.
- Akhir: glitch, vignette, dark room, flash cepat, title/release card.

## File Penting

- `media/*.png`: preview/placeholder scene 1920x1080 untuk import ke CapCut.
- `docs/timeline_capcut.csv`: urutan edit, durasi, efek, transisi, dan cue audio.
- `docs/captions_id.srt`: subtitle yang bisa langsung di-import.

Kalau nanti kamu sudah punya rekaman gameplay asli, ganti media placeholder di CapCut dengan footage kamu, tapi pertahankan durasi dan efeknya.
